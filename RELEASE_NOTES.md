# Release Notları

## macOS v4

- HTML faturada `SAYIN` alanından alıcı adı okunur.
- Alıcı adı çıktı dosyasına yazılır.
- HTML A4’e sığdırma düzeltmesi içerir.
- Kaşe/imza ve çıktı klasörü hatırlanır.

## Windows v1

- Windows için `.exe` build desteği eklendi.
- HTML için Chrome/Edge fallback desteği vardır.
- HTML faturada alıcı adı dosya adına yazılır.
- Kaşe/imza ve çıktı klasörü `%APPDATA%` altında hatırlanır.
